Crie um formulário HTML com um campo de texto para inserir o nome do usuário. Antes de submeter o formulário, verifique se o campo "Nome" está preenchido. Caso contrário, exiba um alerta para o usuário e impeça o envio do formulário. Dica: Utilize o evento onsubmit do formulário para chamar uma função de validação que retorne false se o campo estiver vazio. Document
